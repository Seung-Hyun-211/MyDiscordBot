# MyDiscordBot

|명령어|동작|
|---|---|
|!join|음성 채팅방 입장|
|!p ```url``` or ```검색어```|해당 ```url``` or ```검색어```를 통한 노래 재생|
|!s|현재 재생중인 노래 넘기기|
|!artist ```youtube```|해당 유튜브 채널의 과거 플레이 리스트 재생|
